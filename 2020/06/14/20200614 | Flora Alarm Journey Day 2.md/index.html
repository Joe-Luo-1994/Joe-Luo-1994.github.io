Nullable

以前較不熟悉的部分，針對 NullPointerException 的相關處理

expression description
let Safe Calls: Perform certain operation only for non-null values
?: Elvis Operator: Set default value
?. Safe Calls: Only call property if not null
!! Not-Null Assertion Operator: Conver any value to a non-null type and throws an exception if the value is null
